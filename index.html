Welcome to Null-Sec's GitHub Page.

This page was developed by the members of Null-Sec Staff and its community. It will contain scripts and projects that are commonly used throughout the community and is under constant development.

Installing

Open terminal and make sub-directory for your git clones

mkdir NSTools
cd NSTools
git clone https://github.com/Null-Sec/NSTools.git

Running Scr!pts

Open terminal and go to the scr!pts directory

cd NSTools/Scr!pts/
./{Scr!pt}

Example;

./WPA-BF.py
